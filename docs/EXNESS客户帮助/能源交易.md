**保证金计算方式：**

举例：以 52.934/52.992 的价格买入 5 手美国原油

保证金    =（ 手数 x 合约单位）/杠杆

                = (5 x 1000) / 200 = 25 BBL

根据开盘价将保证金转换为美元（买入订单的买价）

保证金（USD）= 25 x 52.992 = **1324.80 USD**
