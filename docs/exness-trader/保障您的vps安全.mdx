## 保障您的 VPS 安全

虽然拥有交易账户信息即可登录 **VPS**，但 EXNESS 强烈建议您仅将 VPS 用于您自己的交易活动。 与他人共享您的 VPS 登录信息可能会使您的账户和资金面临安全风险和其他潜在威胁。

### 如何查找您的登录信息？

EXNESS 会通过电子邮件向您发送您的 VPS 用户名、密码和服务器的 IP 地址。 EXNESS 建议您妥善保管这些信息，并在收到后立即设置**您自己的密码**。

:::tip 注意
Exness 任何时候都无法登录您的 VPS，也不会知晓您的交易账户密码。
:::
