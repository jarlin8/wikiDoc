# 尊重简单#
第二次世界大战前夕，德国法西斯希特勒为了早日实现他奴役世界的野心，一心想以速战速决的战术赢得战争，但却一直找不到一种高精度、高性能的作战望远镜以配合地面和空中作战。在当时，德国的科学家还没有掌握这种技术。希特勒大为恼火，责令限期研究。 

 德国科学家们心里非常清楚，短期内是不可能研制出这种望远镜的。无奈中，他们只好向英国的同行伸出求助之手。虽然说：“科学无国界”，但在当时那种复杂的环境下，英国科学家自然是“狮子大开口”，要技术资料可以，拿800万美金来。经过一番讨论还价，最终以500万美金成交。 

 德国科学家如获至宝地从英国科学家手中接过技术图纸，打开一看，先是目瞪口呆，随即暴跳如雷，因为这张纸上仅有一条简单的公式而已！一条公式也值500万美金？很快，德国科学家就明白了，这条简单的公式确实值500万美金，因为按公式研制出来的望远镜在闪击波兰的战争中立正了“汗马功劳”。 

 原来，简单也蕴含着巨大的财富。 

 在我们看来，一朵花的绽放、一棵树的发芽、一粒谷的饱满是自然界中再简单不过的现象了，却忽略了它们所经历的冬之侵袭、风之张狂、雨之肆虐。在生活中也是这样，一项技术发明、一件工作的完成，可谓简单，但鲜花背后、荣誉背后却凝聚着工作中的无数日日夜夜和艰辛的劳动。 

 简单往往包含着复杂，就像们的一生。吃饭、睡觉、学习、工作都是很简单的事，可这确实是我们的一生啊。 

 所以，请尊重简单，因为尊重简单就是尊重生命。 

 由此，我们可以判断： 

 生命的意义不是看他的地位、荣誉和财富，而是看其在简单的事情中能折射出多少内在的价值。
