# 不过一碗饭
两个不如意的年轻人，一起去拜见师父。# “师父，我们在办公室被欺负，太痛苦了，求您开示，我们是不是该辞掉工作。”两个年轻人一起问。# 师父闭着眼睛，隔了半天，吐出五个字：  “不过一碗饭。”就挥挥手，示意年轻人退下。# 才回到公司，一个人就递上辞呈，回家种田；另一个却没动。

日子过得真快，转眼十年过去。

回家种田的，以现代方法经营，加上品种改良，居然成了农业专家。

另一个留在公司里，也不差。他忍着气，努力学，渐渐受到器重，已经成为经理。

有一天两个人遇到了。

“奇怪！师父给我们同样五个字，我一听就懂了，不过一碗饭嘛，日子有什么难过，何必硬巴着公司？所以辞职。”农业专家问另一个人，“你当时为什么没听师父的话呢？”

“我听了啊！”那经理笑道：师父说“不过一碗饭”，多受气、多受累，我只要想“不过为了混碗饭吃”，老板说什么是什么，少赌气，少计较，就成了！师父不是这个意思吗？
  两个人又去拜见师父。师父已经很老了，仍然闭着眼睛，隔了半天，答了五个字“不过一念间！”

然后，挥挥手……
